Functions

Some basics javascript/jQuery functions

$('.treeview').treeView('expandAll');

Expands all the nodes of treeviews using the selector i.e. class name treeview in ths case

$('.treeview').treeView('collapseAll');

Collapses all the nodes of treeviews using the selector i.e. class name treeview in ths case

Data Model

TreeViewJS supports generating a treeView from a data model if you don't plan to write some html. The following data generates the treeView below using the code -

$('.treeview').treeView(model);
